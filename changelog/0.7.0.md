# v0.7.0 (2026-02-23)

## 🚀 Features

- **Semantic Memory v2**: Sistema completo de memoria semántica con embeddings (fastembed + usearch)
- **Session Memory Hook**:hook en comando `/new` que guarda contexto de sesión
- **Pre-compaction Flush**: Guarda memoria antes de compactar History
- **Post-compaction Context**: Reinjecta contexto resumido después de compactar
- **Advanced Memory Features**: TTL, citations, MMR (Max Marginal Relevance), temporal decay, query expansion
- **Docs Directory**: `/docs` para archivos del usuario (sesiones, documentos)
- **Wizard Auto-install**: Auto-instala fastembed + usearch cuando se habilita semantic memory
- **Wizard Auto-download**: Auto-descarga modelo de embeddings

## 🐛 Bug Fixes

- **Cron Infinite Loop**: Corregido loop infinito cuando recordatorios creaban más jobs de cron
- **Bot Self-message Detection**: Telegram ahora ignora mensajes del propio bot
- **is_available Lazy Loading**: Memory.is_available ahora dispara lazy loading
- **Cron Cleanup**: Limpia jobs duplicados en startup (41 jobs de "tomar agua" eliminados)

## 🔧 Improvements

- **Semantic Memory Config**: SemanticMemoryConfig con validación de aliases
- **Memory Integration**: Inyección de memoria semántica en ContextBuilder
- **Skills Improvements**: Auto-install de skills, ejemplos, movimiento al inicio del prompt
- **Workspace Organization**: Archivos de sesión en `/docs` en lugar de raíz
