# v0.6.0 (2026-02-22)

## 🚀 Features

- **Skills v2 Motor**: Nuevo sistema de skills con formato XML para prompts
- **skill_router Tool**: El LLM puede consultar qué skill usar basándose en keywords
- **skill_read Tool**: El LLM puede leer el contenido completo de un skill
- **clawhub_install Tool**: El agente puede instalar skills desde el registry de ClawHub
- **Weather Tool con confirmación**: Confirma ubicación antes de dar el clima
- **Auto-learning**: El agente aprende del usuario naturalmente

## 🐛 Bug Fixes

- **Memory consolidation**: Arreglados errores de parsing
- **max_tokens limits**: Límites correctos por modelo
- **Sessions directory**: Limpia correctamente durante onboarding
- **Onboarding**: Mejoras en la detección y flujo

## 🔧 Improvements

- **Skills dinámicos**: Categorías automáticas (_core, _integrations, _tools, _community)
- **Validación de skills**: Validación, truncamiento, emojis, hints de instalación
- **Onboarding mejorado**: El bot actúa "newborn" con identidad neutral
- **Profile.json**: Guarda configuración de CLI
